python笔记（一）

python笔记（一）

六个标准数据类型

  • Number（数字）
  • String（字符串）
  • List（列表）
  • Tuple（元组）
  • Sets（集合）
  • Dictionary（字典）

Number

int、float、bool、complex（复数）

Tips

  • isinstance() & type()

    isinstance()考虑继承关系
    type() 并不考虑 子类与父类不同

  • 除法(/)总返回浮点数，若要获得整数，应使用(//)

String

  • 截取语法：
    print (str) # 输出字符串
    print (str[0:-1]) # 输出第一个到倒数第二个的所有字符
    print (str[0]) # 输出字符串第一个字符
    print (str[2:5]) # 输出从第三个开始到第五个的字符
    print (str[2:]) # 输出从第三个开始的后的所有字符
    print (str * 2) # 输出字符串两次
    print (str + “TEST”) # 连接字符串
  • 转义字符“\”
    若不转义则在字符串前添加R或者r
  • python字符不可改变 即不可重新赋值

List

  • 内部元素可以修改
  • 很像PHP中的数组 可以嵌套 支持多种数据类型

Tuple

  • 使用括号进行书写

运算符

身份运算符

is is not 判断两个标识符是不是来自于同一个对象

遍历序列

a b c d

  • range() 遍历数字序列

  • 因此可以使用range()来循环遍历序列

python迭代器、生成器

迭代器对象从集合的第一个元素开始访问，直到所有的元素被访问完结束。迭代器只能往前不会后退。

生成器是一个返回迭代器的函数，只能用于迭代操作（另一篇文章见）

函数

  • def 函数名（参数列表）:
    函数体
  • python 中一切都是对象，严格意义我们不能说值传递还是引用传递，我们应该说传不可变对象(string,tuples,numbers)或传可变对象(list,dict)。

参数

  • 必需参数
  • 关键字参数 使用关键字参数允许函数调用时参数的顺序与声明时不一致
  • 默认参数 默认参数在定义函数时体现
  • 不定长参数

匿名函数

需要一个函数，但是又不想费神去命名一个函数的场合下使用，也就是指匿名函数

  • lambda的主体是一个表达式，而不是一个代码块
  • 拥有自己命名空间
  • map(function,list)函数常用于此 其作用为对list中每个元素做映射按照function函数

变量作用域

  • L （Local） 局部作用域
  • E （Enclosing） 闭包函数外的函数中
  • G （Global） 全局作用域
  • B （Built-in） 内建作用域
  • 以 L –> E –> G –>B 的规则查找
